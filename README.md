## About OverTheWire

[OverTheWire](https://overthewire.org/wargames/) provides hands-on security wargames covering areas such as:

- Linux and shell fundamentals
- Web security
- Cryptography
- Reverse engineering
- Binary exploitation

These games are designed to build practical problem-solving skills in a safe, educational environment.
